# Productivity Tools for Technical Reporting — Deployed Content

LMS-ready PDFs for the **Productivity Tools for Technical Reporting** course, part of the Operations Support Specialist — Network curriculum (RTI Area 4: Security & Professional Skills).

## Course summary

- **Hours:** 4
- **Modules:** 2
- **Lessons:** 4
- **Activities:** 8 (5 labs + 3 exercises)

## Structure

```
module-NN-[module-slug]/
  lesson-NN-[lesson-slug]/
    lesson-NN-[slug].pdf                     (lesson content — student)
    lesson-NN-instructor-guide.pdf           (instructor-only)
    lesson-NN-quiz.pdf                       (student — no answers)
    lesson-NN-quiz-answer-key.pdf            (instructor — with answers and explanations)
    lesson-NN-[activity-type]-[slug].pdf     (student-facing activity)
    lesson-NN-instructor-guide-[activity-type]-[slug].pdf  (instructor facilitation guide)
```

## Generation

These PDFs are regenerated by running:

```bash
cd _COURSES\ Phase\ 1\ -\ WORKING/design-documentation
node tools/content-dev/deploy/deploy.js \
  --course-slug=productivity-tools-reporting \
  --workspace=../.. \
  --output-dir=../courses/productivity-tools-reporting/deploy
```

Do not edit PDFs directly — edit the markdown source in the design-documentation or courses folder and redeploy.
